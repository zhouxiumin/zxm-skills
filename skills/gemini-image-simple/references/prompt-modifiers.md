# Gemini 图像提示词修饰词速查

常用修饰词整理，可直接附加到任何提示词末尾提升效果。

---

## 风格关键词

| 关键词 | 效果 |
|--------|------|
| `photorealistic` | 照片级写实 |
| `cinematic` | 电影感 |
| `hyperrealistic` | 超写实 |
| `ultra-detailed` | 超精细 |
| `8K resolution` | 8K 高清 |
| `flat design` | 扁平设计 |
| `vector art` | 矢量插画 |
| `isometric` | 等轴测视角 |
| `watercolor` | 水彩风格 |
| `oil painting` | 油画风格 |
| `embroidery texture` | 刺绣质感 |
| `3D cartoon` | 3D 卡通 |
| `chibi` | Q版/小头身 |
| `kawaii` | 可爱日系风 |
| `pixel art` | 像素艺术 |
| `retro` | 复古风 |
| `bioluminescent` | 生物发光 |
| `neon glow` | 霓虹发光 |
| `minimalist` | 极简风 |

---

## 光线

| 关键词 | 效果 |
|--------|------|
| `cinematic lighting` | 电影感打光 |
| `soft studio lighting` | 柔和棚拍光 |
| `dramatic lighting` | 戏剧性光线 |
| `golden hour lighting` | 黄金时段光 |
| `rim lighting` | 轮廓逆光 |
| `lens flare, dof, hdr` | 耀光+景深+高动态 |
| `pure black background` | 纯黑背景（聚焦） |
| `white background` | 纯白背景（产品图） |

---

## 构图

| 关键词 | 效果 |
|--------|------|
| `1:1 ratio` | 正方形 |
| `9:16 ratio` | 竖版（手机屏） |
| `16:9 ratio` | 横版（宽屏） |
| `subject centered, fills the frame` | 主体居中满幅 |
| `front view, slight upward angle (~10 degrees)` | 正面微仰视 |
| `bird eye view` | 俯瞰视角 |
| `close-up` | 特写 |
| `wide shot` | 全景宽镜 |
| `isometric view` | 等轴测视角 |

---

## 材质/质感

| 关键词 | 效果 |
|--------|------|
| `glossy` | 光泽 |
| `matte` | 哑光 |
| `metallic` | 金属 |
| `transparent` / `translucent` | 透明/半透明 |
| `clay material` | 粘土质感 |
| `velvet texture` | 丝绒质感 |
| `glass` | 玻璃 |
| `chrome` | 镜面铬 |
| `holographic` | 全息效果 |
| `frosted glass` | 磨砂玻璃 |

```

```

```
..., neon glow, bioluminescent, dark background, vibrant colors, long exposure effect
```
