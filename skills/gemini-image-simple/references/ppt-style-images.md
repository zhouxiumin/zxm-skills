# Gemini 类 PPT 图片生成参考

Gemini 图像模型（Nano Banana Pro / gemini-image 系列）在文字渲染、信息图表和专业布局方面表现突出，文本准确率 95%+，支持中英文，输出可达 4K，是生成「能直接用」的 PPT 风格图片的最佳选择之一。

---

## 核心原则：像对设计师下指令

不要用关键词堆砌（Tag Soup），要用**完整自然语言**描述意图。

| 弱提示词 ❌ | 强提示词 ✅ |
|-----------|----------|
| 做一个关于 AI 的封面 | 创建一张专业演示文稿封面，标题「AI 战略 2026」，深蓝渐变背景，大号白色字体，右下角极简电路板插图，现代企业风格 |
| 画个流程图 | 创建水平流程图，4 个步骤，步骤间用箭头连接，每个方框内含图标，专业扁平设计 |

**黄金 4 法则：**
1. **对话式修改**：生成后直接说「把背景换成深蓝，标题加大」，不要整体重生成
2. **完整句子描述**：用自然语言，避免碎片词
3. **指定布局细节**：比例、分辨率、颜色、文字密度
4. **风格锚定**：先生成一张母版图，后续所有页上传它作为参考（Image-to-Image）

## 幻灯片类型提示词模板

### 封面幻灯片

```
创建一张专业的演示文稿封面幻灯片，标题为「[主题]」，使用深蓝色渐变背景上的大号白色文字。包含微妙的几何图案和右下角的极简[行业相关]插图。现代企业风格，高对比度，16:9 比例，4K 分辨率。
```

### 数据图表 / 信息图

```
创建一个信息图表。[描述数据，例如：三家公司市场份额：A 公司 32%，B 公司 23%，C 公司 11%]。使用水平条形图，A 公司蓝色，B 公司橙色，C 公司绿色。在每个条形末端标注百分比。标题：[图表标题]。简洁极简设计，白色背景，4K 高清。
```

> 务必提供真实数字——Gemini 能准确定位和标注数据点。

### 流程图

```
创建一个水平流程图，展示 [N] 个步骤：
1）[步骤名]（[颜色]方框）→ 2）[步骤名]（[颜色]方框）→ ...
步骤之间使用箭头连接。每个方框内包含代表各操作的小图标。专业扁平设计风格，白色背景，16:9 比例。
```

> 必须明确写「步骤之间使用箭头」，否则模型可能生成不相连的方框。

### 对比卡片 / 定价表

```
创建一个 [N] 列对比卡片布局：
第 1 列：「[方案名]」，[颜色]标题，功能：[列表]
第 2 列：「[方案名]」，[颜色]标题，功能：[列表]（标注「最受欢迎」徽章）
第 3 列：「[方案名]」，[颜色]标题，功能：[列表]
简洁现代风格，白色背景，清晰分隔线，16:9 比例。
```

### 通用内容页

```
生成一张专业 16:9 比例的 PPT 幻灯片，4K 高清分辨率，[风格描述] 背景，白色/浅灰大字体，文字清晰不密集。

标题：[标题]（居中，大字体）
副标题：[副标题]
左侧：[图表类型]，展示 [数据描述]，[颜色] 配色，数据标签清晰。
右侧：[N] 点关键内容：
• [要点1]
• [要点2]
• [要点3]

整体简洁高端，适合商务汇报，电影级柔和光影，无多余装饰，无水印。
```

## 20 种 PPT / 信息图风格速查

### 第一类：主流通用（安全牌）

| 风格 | 特征 | 适用场景 |
|------|------|----------|
| **国际极简风** (Minimalist) | 大量留白，无衬线粗体，高清大图，无装饰 | 产品发布、路演、高管演讲 |
| **严谨商务风** (Corporate) | 深蓝/灰配色，结构化导航，图表占比高 | 咨询报告、财报、商业计划书 |
| **学术论文风** (Academic) | 白底黑字，宋体/Times New Roman，公式多 | 毕业答辩、科研汇报 |
| **新中式** (New Chinese) | 水墨晕染，书法字体，传统纹样，留白意境 | 节日庆典、文化旅游、党政汇报 |

提示词示例（严谨商务风）：
```
严谨商务风格 PPT 幻灯片，深蓝色(#1E3A8A)和灰色配色，白色文字，结构化布局，图表为主，规整对齐，16:9 比例，4K 分辨率，无装饰边框。
```

### 第二类：现代科技与 UI（流行趋势）

| 风格 | 特征 | 适用场景 |
|------|------|----------|
| **Bento Grid** (便当盒风) | 圆角矩形模块化分区，像便当盒 | 功能概览、目录页、多维数据 |
| **科技未来风** (Tech/Cyberpunk) | 深色背景，发光线条，霓虹渐变，粒子效果 | AI/区块链产品、技术架构图 |
| **毛玻璃风** (Glassmorphism) | 弥散光球背景，半透明磨砂玻璃卡片 | 互联网 App 演示、趋势报告 |
| **粘土风** (Claymorphism) | 3D 充气感图标，软萌马卡龙配色，柔和投影 | 教育、母婴、团建、轻松话题 |
| **暗黑数据风** (Dark Mode Data) | 深灰背景，高亮细线数据图，极客感 | 监控大屏、运维报告、安全汇报 |

提示词示例（毛玻璃风）：
```
毛玻璃 Glassmorphism 风格 PPT 幻灯片，背景为深蓝到紫色 Mesh Gradient 渐变，前景为半透明磨砂玻璃卡片，白色文字，Roboto 字体，16:9 比例，4K 高清，电影级柔和光影。
```

提示词示例（Bento Grid）：
```
Bento Grid 布局幻灯片，页面分为 [N] 个圆角矩形区块，[颜色]配色，每块展示独立信息模块，模块间有间隙，整体现代简洁，16:9 比例。
```

### 第三类：艺术与创意（视觉冲击）

| 风格 | 特征 | 适用场景 |
|------|------|----------|

| **拼贴风** (Collage) | 撕纸边缘，胶带效果，多材质堆叠 | 旅游日记、创意头脑风暴 |
| **噪点渐变风** (Grainy Gradient) | 模糊梦幻渐变，颗粒噪点覆盖，复古科技感 | 时尚品牌、年轻化产品 |

### 第四类：特殊功能（解决特定问题）

| 风格 | 特征 | 适用场景 |
|------|------|----------|
| **2.5D 等距视角** (Isometric) | 上帝视角 45°，无透视灭点，矢量插画 | 物流供应链、智慧城市、IT 架构图 |
| **手绘涂鸦风** (Hand-Drawn) | 抖动线条，简笔画，箭头圈注，纸张纹理 | 教育培训、创业路演、故事叙述 |
| **大字报风** (Big Type) | 字号极大撑满屏幕，排版夸张，极简色彩 | 宣言页、金句页、核心价值观 |
| **3D 数据可视化** (Immersive Data) | 图表立体化（建筑状柱图），动感 | 年度战报、物流数据分析 |

提示词示例（2.5D 等距）：

```
2.5D 等距视角插画风格幻灯片，上帝视角 45°，矢量扁平插画，展示 [场景描述：如智慧城市/IT架构]，清晰线条，柔和配色，白色背景，16:9 比例。
```

## 风格前缀：保持多页一致性

为多页 PPT 保持视觉统一，先定义风格前缀，每次生成时都加在提示词最前面：

```
[风格前缀示例] 企业风格。海军蓝(#1E3A8A)背景，白色文字，Roboto 字体，极简设计，无装饰边框，16:9 比例，4K 分辨率。
```

建议将前缀保存在团队文档中，所有人使用同一前缀，确保整套 PPT 视觉一致。

## 负向提示词：避免不想要的元素

在提示词末尾加入否定指令，防止常见问题：

| 问题 | 加入否定指令 |
|------|--------------|
| 出现水印 | `无水印` |
| 装饰过度 | `无装饰边框，无多余元素` |
| 投影太重 | `无阴影` |
| 消费级美学 | `简洁、专业、极简风格` |
| 文字模糊 | `文字清晰，高对比度` |

**通用否定后缀：**
```
无水印，无装饰边框，无多余装饰，文字清晰，简洁专业，极简风格。
```
