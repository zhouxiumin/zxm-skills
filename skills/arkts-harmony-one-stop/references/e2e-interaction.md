# 模拟交互

在模拟器中模拟用户点击和滑动操作。

## 前提条件

1. 能从截图中识别目标控件的大致坐标。
2. 明确设备分辨率与坐标系（左上角为原点 `(0, 0)`）。

## 点击操作

### 基本语法

```bash
hdc shell uinput --touch --click <x> <y>
```

### 示例

```bash
hdc shell uinput --touch --click 658 2600
```

## 滑动操作

### 基本语法

```bash
hdc shell uinput -T -g <起始x> <起始y> <结束x> <结束y> <持续时间1> <持续时间2>
```

参数说明：

- `<起始x> <起始y>`: 起始坐标
- `<结束x> <结束y>`: 结束坐标
- `<持续时间1>`: 通常 500（毫秒）
- `<持续时间2>`: 通常 1000（毫秒）

### 示例

假设设备尺寸为 `466x466`：

#### 向上滚动

```bash
hdc shell uinput -T -g 233 233 233 0 500 1000
```

#### 向下滚动

```bash
hdc shell uinput -T -g 233 0 233 233 500 1000
```

## 使用流程

1. 截图获取当前 UI
2. 确定目标区域坐标
3. 执行点击/滑动命令
4. 再次截图或查看日志确认结果

## 注意事项

- 坐标越界会导致操作无效
- 不同设备分辨率不同，坐标需按设备调整
- 若操作不生效，先确认目标设备在线且在前台页面
