# 小说大纲结构指南

## 三幕结构（适合都市小说）

都市小说推荐使用三幕结构，清晰、灵活，适合各种长度和类型。

### 结构比例

```
第一幕：建立世界（20-25%）
第二幕：对抗与成长（50-60%）
第三幕：高潮与结局（15-20%）
```

---

## 第一幕：建立（Setup）

**目标**：介绍角色、世界、核心冲突

### 关键节点

#### 1. 开场（Opening）
- **位置**：前1-3%
- **功能**：抓住读者注意力，展示故事基调
- **都市小说开场技巧**：
  - 从日常切入：上班路上、咖啡馆、公司会议
  - 从冲突切入：接到电话、发现秘密、意外相遇
  - 从情绪切入：迷茫、焦虑、期待

**示例**：
```
❌ 避免：林晓是一个普通的白领，今年27岁...
✅ 推荐：林晓盯着电脑屏幕上的辞职信，手指悬在发送键上，已经三个小时了。
```

#### 2. 日常世界（Ordinary World）
- **位置**：前10-15%
- **功能**：展示主角的初始状态和问题
- **展现内容**：
  - 主角的工作生活
  - 人际关系现状
  - 内心渴望与困境
  - 价值观和行为模式

**注意**：不要花太长时间在日常世界，现代读者期待快速进入冲突。

#### 3. 激励事件（Inciting Incident）
- **位置**：10-15%
- **功能**：打破日常，引入主要冲突
- **都市小说常见激励事件**：
  - 职场：被辞退、升职机会、新项目
  - 情感：旧爱重逢、意外邂逅、关系破裂
  - 家庭：亲人变故、秘密揭露
  - 自我：梦想机会、身份危机

**要求**：激励事件要有足够的冲击力，让主角无法回到原来的生活。

#### 4. 第一幕转折点（Plot Point 1）
- **位置**：20-25%
- **功能**：主角做出决定，正式进入新世界
- **特征**：
  - 主动选择（而非被动接受）
  - 不可逆转（无法回头）
  - 引出第二幕的核心冲突

**示例**：
- 接受新工作
- 开始一段关系
- 决定改变自己
- 采取重大行动

---

## 第二幕：对抗（Confrontation）

**目标**：主角面对障碍、成长、失败、再尝试

### 关键节点

#### 5. 新世界适应（Fun and Games）
- **位置**：25-40%
- **功能**：主角探索新环境，享受初步成功
- **都市小说展现**：
  - 新工作的新鲜感
  - 恋爱的甜蜜期
  - 计划的顺利推进
  - 小胜利和成就感

**节奏**：这部分应该是相对轻松的，让读者和主角一起享受变化。

#### 6. 中点（Midpoint）
- **位置**：50%（整个故事的正中间）
- **功能**：重大转折，改变故事方向
- **两种类型**：
  - **虚假胜利**：看似成功，但引出更大问题
  - **重大失败**：遭遇挫折，必须重新规划

**中点的重要性**：
- 提升赌注（Stakes）
- 增加紧迫感（Urgency）
- 揭示新信息（Revelation）
- 迫使主角改变策略

**都市小说中点示例**：
- 情感：表白成功 → 发现对方秘密
- 职场：获得晋升 → 卷入权力斗争
- 创业：融资成功 → 合伙人背叛

#### 7. 一切皆失（All is Lost）
- **位置**：75%
- **功能**：主角经历最低点，失去重要的东西
- **失去的可能是**：
  - 关系（恋人、朋友、家人）
  - 目标（工作、项目、梦想）
  - 信念（对自己或他人的信任）
  - 身份（社会地位、自我认知）

**情感低点**：这是主角最绝望的时刻，也是读者最揪心的时刻。

#### 8. 黎明前的黑暗（Dark Night of the Soul）
- **位置**：75-80%
- **功能**：主角内心挣扎，寻找答案
- **展现方式**：
  - 独自反思
  - 寻求建议
  - 回忆过往
  - 顿悟时刻

**关键**：主角意识到真正需要改变的是自己，而不是外部世界。

#### 9. 第二幕转折点（Plot Point 2）
- **位置**：80%
- **功能**：主角找到解决方案，重新出发
- **特征**：
  - 获得新认知
  - 制定新计划
  - 重拾勇气
  - 准备最后一战

---

## 第三幕：解决（Resolution）

**目标**：最终对决，完成角色弧光

#### 10. 高潮（Climax）
- **位置**：85-95%
- **功能**：主角面对最大挑战，运用所学
- **都市小说高潮**：
  - 不一定是打架或激烈冲突
  - 可以是重要对话
  - 可以是关键选择
  - 可以是公开表态

**高潮要求**：
- 主角主动行动（而非被动应对）
- 运用在故事中获得的成长
- 体现主题和价值观
- 给读者情感宣泄

#### 11. 结局（Resolution）
- **位置**：95-100%
- **功能**：展示新的平衡，呼应开头
- **展现内容**：
  - 关系的新状态
  - 主角的内在改变
  - 世界的新秩序
  - 未来的可能性

**注意**：
- 不要拖沓，高潮后快速收尾
- 留白比说满更有韵味
- 可以留悬念，但主线要有交代

## 情节线索管理

### 主线与支线

```
主线（Main Plot）
├── 情感线（如果是重点）
├── 职场线
└── 成长线（内在变化）

支线（Subplots）
├── 次要关系
├── 背景设定
└── 主题深化
```

### 多线交织原则

1. **支线服务主线**：每条支线都应该推动或影响主线
2. **交替推进**：避免长时间只写一条线
3. **关键处汇合**：在高潮时，多条线索共同作用
4. **独立完整**：每条线都有自己的起承转合

## 节拍法（Save the Cat 方法）

适合追求紧凑节奏的都市小说，更细致的15个节拍：

1. **开场意象**（0-1%）：定调
2. **主题提示**（5%）：提出核心问题
3. **铺垫**（1-10%）：日常世界
4. **推动**（10%）：激励事件
5. **争执**（10-20%）：犹豫不决
6. **突破第二幕**（20%）：做出决定
7. **副线**（22%）：引入感情线或次要冲突
8. **玩乐时刻**（20-50%）：故事承诺的乐趣
9. **中点**（50%）：虚假胜利或重大失败
10. **坏人进逼**（50-75%）：压力增加
11. **一切皆失**（75%）：最低点
12. **灵魂黑夜**（75-80%）：内心挣扎
13. **突破第三幕**（80%）：找到答案
14. **终场**（80-99%）：最终对决
15. **终场意象**（99-100%）：呼应开场

## 使用建议

### 提纲派写作流程

1. 先完成完整大纲（包括所有关键节点）
2. 列出所有必写场景
3. 每个场景写简要梗概（3-5句话）
4. 按顺序扩写每个场景
5. 写完后再整体调整节奏

### 混合派写作流程

1. 完成三幕结构框架（只到关键转折点）
2. 写出前几章，感受角色和风格
3. 根据实际写作调整后续大纲
4. 边写边细化未写部分的大纲
5. 定期回顾整体结构，防止跑偏

### 大纲常见问题

**问题1：大纲太僵硬，限制了创作**
- 解决：把大纲当成GPS，可以绕路，但要知道终点

**问题2：大纲太详细，失去写作动力**
- 解决：只规划到关键转折点，细节留给写作时发现

**问题3：写着写着偏离大纲**
- 解决：问自己：新方向更好吗？如果是，更新大纲；如果不是，回到原计划

**问题4：不知道每幕该写多少字**
- 参考：
  - 10万字小说：2.5万 / 5.5万 / 2万
  - 20万字小说：5万 / 11万 / 4万
  - 30万字小说：7.5万 / 16.5万 / 6万

### 检查大纲质量

好的大纲应该能回答：

- [ ] 主角的欲望和恐惧清晰吗？
- [ ] 每个转折点都有足够的冲击力吗？
- [ ] 主角有成长弧光吗？
- [ ] 冲突逐步升级了吗？
- [ ] 结局回应了开头吗？
- [ ] 支线服务主线了吗？
- [ ] 每一幕的功能明确吗？
- [ ] 节奏有张弛吗？

如果以上问题都是"是"，那么大纲已经很扎实了。
