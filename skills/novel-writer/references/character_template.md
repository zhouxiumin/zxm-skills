## 使用指南

### 创建新角色时

1. **主角**：填写完整模板，确保立体丰满
2. **重要配角**：填写80%，省略不影响情节的细节
3. **次要角色**：只填写基础信息、性格特质、关系网络
4. **功能性角色**：只需要基础信息和一句话性格描述

### 角色设定原则

1. **有缺点才真实**：完美的角色是扁平的，缺点让角色可信
2. **欲望驱动行动**：明确角色想要什么，行为才有逻辑
3. **背景决定反应**：角色的过去塑造了TA对事件的反应方式
4. **关系产生张力**：角色间的关系是冲突和情节的来源
5. **成长才有弧光**：让角色在故事中改变，而不是保持静态

### 都市小说角色特点

**职业设定要真实**：
- 研究具体行业的工作内容和术语
- 避免"霸道总裁"等脸谱化职业
- 展现职业对角色性格和生活的影响

**现代生活细节**：
- 社交媒体使用习惯
- 通勤方式
- 消费观念
- 娱乐方式

**情感关系复杂性**：
- 不只是爱情，还有友情、职场关系、家庭关系
- 现代人的情感困境：工作压力、原生家庭、自我实现

### 角色一致性检查表

在续写过程中，定期检查：

- [ ] 角色的反应符合其性格设定吗？
- [ ] 角色的说话方式保持一致吗？
- [ ] 角色的能力边界没有突然改变吧？
- [ ] 角色对其他人的态度符合关系设定吗？
- [ ] 角色的价值观在选择中体现了吗？
- [ ] 角色有没有"看起来很厉害"但实际很空洞？

### 避免的角色陷阱

1. **完美角色**：没有缺点 = 没有成长空间
2. **功能性存在**：只为推动情节而存在，没有独立人格
3. **前后矛盾**：性格突变没有铺垫
4. **脸谱化**：霸道总裁、傻白甜、工具人配角
5. **信息堆砌**：一次性倾倒所有背景信息
6. **过度巧合**：角色恰好有剧情需要的每一项技能

### 角色关系动态图示例

```
主角 ←--暗恋-- 配角A
  ↓                ↑
竞争              师徒
  ↓                ↑
配角B --好友--> 配角C
```

创建角色时，画出关系图，能更清楚地看到冲突和互动的可能性。
